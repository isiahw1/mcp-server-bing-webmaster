Initial commit: MCP Server for Bing Webmaster Tools

A comprehensive Model Context Protocol (MCP) server providing direct access 
to all 59 Bing Webmaster Tools API endpoints with 100% API coverage.

Features:
- Full MCP specification compliance
- OData v2/v3 response format handling
- Comprehensive error handling and logging
- Type-safe implementation with annotations
- Cross-platform support (Windows, macOS, Linux)

This package enables programmatic access to Bing Webmaster Tools through
Claude and other MCP-compatible AI assistants.
